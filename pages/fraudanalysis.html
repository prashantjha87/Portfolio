Fraud Analysis

Goal

The goal of this project was to find the end-to-end transaction flow, this includes finding the intermediate accounts which involve in that transaction.In this, we had given starting accounts from which the transaction initiated and the date of the transactions.

Result

To achieve this goal we had to fetch the data from the data warehouse with a certain condition (i.e Customer Number and Transaction Date)Once data is fetched then we did some data quality checks.Once we had the clean data we created a data mart(we have used SAS Data Studio for data mart creation)After that, we created a Sankey Diagram to find out the transaction flow between different accounts in the given period.
